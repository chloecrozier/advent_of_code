# [Advent Of Code](https://adventofcode.com)

1. Set your AoC your session ID as a variable in an .env file: \
&nbsp;`AOC_SESSION_ID='{YOUR_SESSION_ID}'`

2. Save the input as a .txt file, then run: \
&nbsp;`python get_input_data.py {YEAR} {DAY}`

3. Run your solution scripts with the input data: \
&nbsp;`python day{#}.py {INPUT_PATH}`
